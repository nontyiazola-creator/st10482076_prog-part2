# Overview of the Bot 
The Cybersecurity Awareness Chatbot is a WPF desktop application developed using C# and .NET 8 in Visual Studio 2022. The chatbot was created to help users learn about common cybersecurity threats and safe online practices through interactive conversations.

The application allows users to ask questions related to cybersecurity topics such as phishing, malware, password safety, VPN security, ransomware, and online scams. The chatbot responds using a keyword recognition system with multiple randomised responses to create more natural interactions.

The chatbot also includes sentiment detection functionality. It can identify when a user is worried, curious, or frustrated and automatically provide useful cybersecurity advice and tips. This improves user engagement and creates a more supportive learning experience.

A memory system was implemented to store and recall the user’s name during the conversation, allowing the chatbot to personalise responses. The application also supports follow-up conversation flow using phrases such as “tell me more” and “explain more” to continue discussing the previous topic.

The graphical user interface was developed using Windows Presentation Foundation (WPF) and includes a chat display area, input textbox, Send button, ASCII art banner, ScrollViewer, and voice greeting functionality using a greeting.wav audio file.

The project follows object-oriented programming principles by separating functionality into different classes:

ChatBot.cs
KeywordResponder.cs
SentimentDetector.cs
MemoryStore.cs

GitHub Actions Continuous Integration (CI) was also implemented to automatically verify successful project builds after code updates.
